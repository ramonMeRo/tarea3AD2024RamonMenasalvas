Ayuda - Panel de Administrador

Funciones del Administrador

Desde esta pantalla, puedes gestionar los usuarios y las paradas del sistema.

Registrar una nueva parada

Para registrar una nueva parada, sigue estos pasos:

  1. Introduce el nombre de la parada.
  2. Escribe la región donde se encuentra la parada.
  3. Ingresa los datos del usuario responsable (nombre, email y contraseña).
  4. Haz clic en el botón "Guardar".

Si la parada ya existe, se mostrará un mensaje de advertencia.

Editar un usuario

Para modificar la información de un usuario:

  1. Selecciona el usuario en la tabla.
  2. Modifica los datos en los campos correspondientes.
  3. Haz clic en "Guardar" para aplicar los cambios.

Eliminar un usuario

Para eliminar un usuario, selecciona la fila en la tabla y haz clic en la opción "Eliminar usuario".

Volver al inicio

Para regresar a la pantalla principal, haz clic en "Salir".
