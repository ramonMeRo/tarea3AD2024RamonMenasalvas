Ayuda - Responsable Controller

¿Cómo registrar un peregrino?

Para registrar el sellado del carnet de un peregrino, sigue estos pasos:

  1. Selecciona un peregrino de la lista desplegable "Seleccionar Peregrino".
  2. Indica si el peregrino va a quedarse en la parada, marcando "Sí" o "No" en la opción "Estancia".
  3. Si el peregrino se queda, elige si es VIP marcando "Sí" o "No" en la opción "VIP".
  4. Haz clic en "Sellar Carnet" para registrar la visita y actualizar el carnet del peregrino.

¿Qué información se requiere?

Para sellar el carnet de un peregrino, se requiere la siguiente información:

  • Nombre del peregrino: Se autocompletará cuando selecciones el peregrino de la lista.
  • Apellidos del peregrino: También se autocompletará automáticamente.
  • Fecha de nacimiento: Información del peregrino ya registrada en su perfil.
  • Estancia: Indicar si el peregrino se queda o no en la parada.
  • VIP: Si el peregrino es VIP, se marcará esta opción.

¿Qué pasa si el peregrino ya ha sellado el carnet hoy?

Si el peregrino ya ha sellado el carnet el día de hoy, se mostrará un mensaje de error indicando que el peregrino no puede sellar nuevamente. Verifica si la fecha de la última visita coincide con el día actual.

Recuerda que un peregrino solo puede sellar una vez por día.

¿Cómo exportar las estancias de un peregrino?

Para exportar las estancias de un peregrino dentro de un rango de fechas:

  1. Selecciona las fechas de inicio y fin usando los campos "Fecha Inicio" y "Fecha Fin".
  2. Haz clic en "Exportar Parada".
  3. El sistema mostrará todas las estancias del peregrino dentro del rango de fechas seleccionado.
